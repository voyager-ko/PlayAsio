## 注意
今回利用するASIO / ASIO4ALLは，Windows向けの汎用ドライバであるため，Windows環境内で実装してください．

# PlayAsio

PlayAsioは，ASIO4ALLを用いてリアルタイム音声入出力を行い，入力音声にエコー処理を加えて出力するC/C++プロジェクトです。

本プロジェクトでは，低遅延な音声処理を実現するためにASIOドライバを利用し，マイクなどから取得した音声データに対してリアルタイムでエコー処理を行います。

## 概要

一般的な音声処理では，音声ファイルを読み込んで処理を行うオフライン処理があります。一方で，本プロジェクトでは，入力された音声をその場で処理し，すぐに出力するリアルタイム処理を対象としています。

具体的には，ASIO4ALLを用いて音声デバイスから入力データを取得し，取得した音声データにエコー処理を加えた後，スピーカーやイヤホンへ出力します。

## 実装内容

本プロジェクトでは，主に以下の処理を実装しています。

- ASIO4ALLを用いた音声デバイスの制御
- ASIOのコールバック関数によるリアルタイムエコー処理
- 処理後の音声データのリアルタイム出力
- 実行後の音声ファイルの記録

## 実装方法
1. Visual Studio上でPlayAsioをクローンします．
2. クローンしたフォルダ内には，local_libというフォルダがあるのでそのフォルダを、自身のC:\に配置します．
3. 最後に，ライブラリが準備されているかを調べるために，Visual StudioのPlayAsioのプロパティを開きます．次に，C/C++　→　全般　→　追加のインクルードディレクトリの中がlocal_libのフォルダを参照しているかのチェックを行います．
   また，リンカー　→　全般　→　追加のライブラリディレクトリにもlocak_lib内のlibファルダが参照されているかのチェックを行います．最後に，リンカー　→　入力　→　追加の依存ファイルにlibfftw3-3.lib，sndfile.libが書いてあるかを
   確認します．
4. Windows上で低遅延な音声入出力を行うための汎用ドライバをインストールします．https://asio4all.org/のサイトにアクセスを行いv2.16をインストールします．
5. ASIO4ALLを開き、自分の利用したいマイクとスピーカーの選択を行い，Visual Studioから実行を行うとリアルタイムでエコー処理がかかるようになっています．また，最初の実装では，現在ステレオの実装に問題が発生しているため、モノラル出力になっています．
6. 最後に，実行中に話された内容はresult.wavとして保存されています．
